#  Cek & Balance

Sub Repositori ini akan berisi kode sumber untuk aplikasi Cek & Balance sebuah pemaparan dana pensiun per 1 maret 2025 dari sisi akademis, yang membahas tentang Usulan Rencana Dana Pensiun Indonesia (DPI) untuk meningkatkan kesejahteraan pensiunan.

## Struktur Repositori
- `src/`: Berisi kode sumber aplikasi.
- `docs/`: Dokumentasi terkait aplikasi.
- `tests/`: Kode untuk pengujian aplikasi.
- `README.md`: File ini, berisi informasi tentang repositori.
- `LICENSE`: Lisensi untuk repositori ini.
- `data/`: Berisi data yang digunakan dalam aplikasi.

## Instalasi
1. Clone repositori ini: 
   ```bash
   git clone
   ```
2. Masuk ke direktori proyek:
   ```bash
    cd cek-balance
    ```
3. Instal dependensi yang diperlukan:
    ```bash
    pip install -r requirements.txt
    ```
